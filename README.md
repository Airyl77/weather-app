# Weather App

A simple React app built with Vite that lets you look up current weather conditions for any city. It also showcases interactive profile cards with a like counter.

## Features

- Search current temperature and weather description for any city via the [wttr.in](https://wttr.in) API
- Interactive `ProfileCard` components with a like button
- Fast development workflow with Vite and HMR

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Scripts

| Command           | Description                  |
| ----------------- | ---------------------------- |
| `npm run dev`     | Start the development server |
| `npm run build`   | Build for production         |
| `npm run preview` | Preview the production build |
| `npm run lint`    | Run ESLint                   |

## Tech Stack

- [React 19](https://react.dev)
- [Vite](https://vite.dev)
- [wttr.in](https://wttr.in) — weather data API
